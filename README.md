# Temperature Prediction Using Linear Regression

## Overview

This project predicts temperature based on humidity values using a Machine Learning Linear Regression model. The objective is to analyze the relationship between humidity and temperature and build a predictive model using Python and Scikit-Learn.

## Project Objectives

* Perform data analysis on humidity and temperature data.
* Visualize the relationship between humidity and temperature.
* Build a Linear Regression model for temperature prediction.
* Evaluate model performance using standard metrics.

## Dataset

The dataset contains humidity and temperature observations used for training and testing the machine learning model.

### Features

* **Humidity** – Input feature

### Target Variable

* **Temperature** – Predicted output
